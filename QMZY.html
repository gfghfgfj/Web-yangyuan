JavaScript核心语法

JavaScript基本概念

JavaScript是一种轻量级的解释型编程语言，用于创建交互式网页。它包含以下核心概念：

  • 变量声明：使用let、const和var声明变量
  • 数据类型：包括数字、字符串、布尔值、对象等
  • 运算符：算术、比较和逻辑运算符
  • 控制结构：条件语句和循环语句
  • 函数：可重复使用的代码块
// 变量声明与数据类型 let a = 100; const api_key = 'ssssx-=-dddd'; console.log(a); console.log(typeof a, typeof api_key, typeof true);

控制台输出

变量声明

使用let声明块级作用域变量，const声明常量。变量名区分大小写，支持Unicode字符。

类型转换

JavaScript是弱类型语言，支持隐式和显式类型转换。使用Number()、String()等函数进行显式转换。

相等比较

== 运算符进行类型转换后的比较，=== 运算符进行严格比较（值和类型都相同）。

// 类型转换与比较 let b = '1'; console.log(a + b); // 字符串拼接 console.log(a + Number(b)); // 数值相加 let c = '100'; console.log(a == c); // true (值相等) console.log(a === c); // false (类型不同)

控制台输出

控制结构

控制结构用于控制代码的执行流程：

  • 条件语句：if...else，switch
  • 循环语句：for，while，do...while
  • 跳转语句：break，continue，return
// 条件语句 let current_time = '12:20'; if (current_time == '12:20') { console.log('冲啊！'); } else { console.log('假装学习……'); } // for循环 for (let i = 1; i <= 10; i++) { console.log(i); } // while循环 let hug = 60; while (hug > 0) { console.log('继续干饭……'); hug = hug - 10; }

控制台输出

循环控制

for循环适合已知迭代次数的情况，while循环适合条件控制的情况。break可跳出循环，continue跳过当前迭代。

条件判断

if语句根据条件执行不同代码块，可配合else if处理多个条件。三元运算符?:是简写的条件表达式。

函数

函数是JavaScript的基本构建块，用于封装可重用代码：

  • 函数声明：使用function关键字定义
  • 参数：函数可以接受零个或多个参数
  • 返回值：使用return语句返回值
  • 作用域：函数创建自己的作用域
// 函数定义与调用 function intro() { console.log('你好，我是一个脚本！'); } intro(); function intro2(content) { console.log('以下是我介绍的内容： ' + String(content)); } intro2('这是传入的参数'); function intro3(content1, content2) { console.log('以下是我介绍的内容： ' + content1 + content2); } intro3('参数1', '参数2'); function intro4(content) { console.log(content); return content + '这是返回值'; } let result = intro4('带返回值的函数'); console.log(result);

异步编程基础

JavaScript是单线程语言，异步编程允许我们在等待耗时操作（如网络请求）完成时执行其他任务。

setTimeout示例

setTimeout(() => { console.log('111'); }, 2000); console.log('222');

输出顺序：先输出222，2秒后输出111

Promise示例

function getSomething() { return new Promise((resolve) => { setTimeout(() => { console.log('1111'); resolve(); }, 1000) }) } getSomething().then(() => { console.log('2222'); });

输出顺序：1秒后输出1111，然后立即输出2222

async/await语法

async/await是ES7引入的语法糖，让异步代码看起来像同步代码，提高了可读性。

async/await示例

async function getSomething2() { await new Promise((resolve) => { setTimeout(() => { console.log('111111'); resolve(); }, 1000) }); } (async () => { await getSomething2(); console.log('222222'); })();

输出顺序：1秒后输出111111，然后立即输出222222

GitHub API实战

使用Fetch API获取GitHub仓库的提交记录，展示异步数据获取能力。

获取GitHub提交记录

async function getAllCommitDates(owner, repo) { const url = `https://api.github.com/repos/${owner}/${repo}/commits`; try { const response = await fetch(url); if (!response.ok) { throw new Error('网络响应失败'); } const commits = await response.json(); // 提取每个提交的日期 const dates = commits.map(commit => commit.commit.author.date); return dates; } catch (error) { console.error('获取提交日期失败:', error); return []; } }

此函数获取指定仓库的所有提交日期，返回一个日期数组。
